# 删除有序数组中的重复项 II

原题链接 (opens new window) 给你一个有序数组 nums ，请你 原地 删除重复出现的元素，使得出现次数超过两次的元素只出现两次 ，返回删除后数组的新长度。 不要使用额外的数组空间，你必须在 原地 修改输入数组 并在使用 O(1) 额外空间的条件下完成。

示例 1：

输入：nums = [1,1,1,2,2,3] 输出：5, nums = [1,1,2,2,3] 解释：函数应返回新长度 length = 5, 并且原数组的前五个元素被修改为 1, 1, 2, 2, 3。 不需要考虑数组中超出新长度后面的元素。

示例 2：

输入：nums = [0,0,1,1,1,1,2,3,3] 输出：7, nums = [0,0,1,1,2,3,3] 解释：函数应返回新长度 length = 7, 并且原数组的前七个元素被修改为 0, 0, 1, 1, 2, 3, 3。不需要考虑数组中超出新长度后面的元素。

提示：

  • 1 <= nums.length <= 3 * 104
  • -104 <= nums[i] <= 104
  • nums 已按升序排列

# 题解

只需要判断当前元素是否和之前队列末尾两个元素是否同时相等。

class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int m = nums.size();
        if (m <= 2) {
           return m;
        }
        int pos = 2 ;
        for (int i = 2; i < m; i++){
            if (nums[i] != nums[pos - 1] || nums[i] != nums[pos - 2]) {
                 nums[pos++] = nums[i];
            }
        }
        return pos;
    }
};
