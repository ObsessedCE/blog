# 三数之和

原题链接 (opens new window) 给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例 1：

输入：nums = [-1,0,1,2,-1,-4] 输出：[[-1,-1,2],[-1,0,1]] 解释： nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。 nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。 nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。 不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。 注意，输出的顺序和三元组的顺序并不重要。

示例 2：

输入：nums = [0,1,1] 输出：[] 解释：唯一可能的三元组和不为 0 。

示例 3：

输入：nums = [0,0,0] 输出：[[0,0,0]] 解释：唯一可能的三元组和为 0 。

提示：

  • 3 <= nums.length <= 3000
  • -105 <= nums[i] <= 105

# 题解

双指针。三数之和确定一个值 => 两数之和。重点是如何解决去重问题。

class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> res;
        sort(nums.begin(), nums.end());
        int n = nums.size();
        for (int i = 0; i < n - 1; i++) {
            // 保证第一个值不重复
            if (i > 0 && nums[i] == nums[i-1]) {
                continue;
            }
            int left = i+1, right = n -1; 
            while (left < right) {
                // left = i + 1值相同是可以选择的。 
                //  保证中间枚举left和right重复，只需要保证一个不重复，结果中两个就不会重复 a + b + c = n, a是确定的，b 不重复那么c也不会重复
                if (left -1 != i && nums[left] == nums[left-1]) {
                    left++;
                    continue;
                }
                int tmp = nums[i] + nums[left] + nums[right];
                if (tmp > 0) {
                    right--;
                } else if (tmp < 0){
                    left++;
                } else {
                    res.push_back({nums[i] , nums[left] , nums[right]});
                    left++;
                    right--;
                }
            }
        }
        return res;
    }
};
